# Portfolio Website — Vonn Adrian Jutar

Personal portfolio built with React 19, TypeScript, TailwindCSS, and Vite.

## Tech Stack

- **Framework:** React + TypeScript
- **Build tool:** Vite
- **Styling:** TailwindCSS
- **Fonts:** Playfair Display (headings), Inter (body)
- **Icons:** react-icons
- **Animation:** react-type-animation
- **Formatting:** ESLint + Prettier

## Features

- Mobile friendly responsive single-page layout
- Sections: About, Experience, Skills, Projects, Education, Contact
- Fully typed components and data (makes it easier to add more info)
- Docker containerization (more for learning purposes)
- CI/CD via GitHub Actions (deploys to GitHub Pages)
